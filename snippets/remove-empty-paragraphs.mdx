---
title: Remove empty paragraphs
date: 15.05.2023
description: Renders an accordion menu with multiple collapsible content elements.
category: javascript
---

Remove empty paragraphs space (jQuery):

```js
$('p').each(function() {
     var $p = $(this);
     if($.trim($p.html())==='') {
        $p.remove();
     }
});
```
