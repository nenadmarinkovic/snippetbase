Alter time, you can specify the commit date, in case you need to fill a gap or two in your commit streaks.

```bash
GIT_AUTHOR_DATE='Fri Jul 26 19:32:10 2013 -0400' GIT_COMMITTER_DATE='Fri Jul 26 19:32:10 2013 -0400' git commit

```
