---
title: Change email and username
date: 04.09.2021
description: Configure an individual repo to use a specific user.
category: git

---

Configure an individual repo to use a specific user / email address which overrides the global configuration. From the root of the repo, run:

```bash

git config user.name "Your Name Here"
git config user.email your@email.com

```

Or globally:

```bash

git config --global user.name "Your Name Here"
git config --global user.email your@email.com

```
