---
title: Allow submit only checked checkboxes
date: 01.09.2022
description: Enable or disable a submit button based on the state of multiple checkboxes
category: javascript
---

Allow submit only when checkboxes are checked. Also add onclick event on HTML element: onclick=check()


```js
function check(element) {
  var checkbox1 = document.getElementById("id1");
  var checkbox2 = document.getElementById("id2");
  var submitButton = document.getElementById("submitForm");

  if (checkbox1.checked == true && checkbox2.checked == true) {
    submitButton.disabled = false;
 
  } else {
    submitButton.disabled = true;
  }
}
```
